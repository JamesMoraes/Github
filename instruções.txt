1) Transformar a aplicação web
para desktop

a) Instalar o pacote 'pywebview' e 'mysqlclient'
- Comando no Terminal: pip install pywebview mysqlclient

b) Criar o arquivo 'app_desktop.py'
na pasta raiz do projeto
- Copiar todo o código do arquivo 'app.py'
para 'app_desktop.py'

c) Seguir as instruções dos comentários
do arquivo 'app_desktop.py'

d) Executando o arquivo 'app_desktop.py'
- Comando no Terminal: python app_desktop.py

e) Gerando um arquivo executável

- Instalar o pacote 'cx_Freeze' com o
comando no Terminal: pip install cx_Freeze

- Para gerar o arquivo executável, digitar
o comando no Terminal: cxfreeze app_desktop.py --target-dir dist






2) Transformar a aplicação em um PWA
(Progressive Web Aplication)

a) Criando a pasta 'static' na pasta
raiz do projeto

b) Criando os arquivos 'manifest.json'
e 'service-worker.js' dentro da pasta
'static'
